✅ Lo que SÍ incluye:

✓ Plataforma web completa con registro, login y dashboard personalizado (responsive)
✓ Gestión de documentos: El usuario puede subir hoja de vida, pruebas psicotécnicas (MBTI, iStrong, Big 5), diplomas, certificados
✓ Motor de IA que analiza los documentos subidos: Lee y procesa exámenes psicotécnicos, extrae personalidad, fortalezas e intereses
✓ Tests internos gratuitos: RIASEC (integración O*NET), evaluación de inglés, cuestionario de intereses
✓ Construcción progresiva de CV: El usuario carga logros, habilidades y experiencias dentro de la plataforma
✓ Motor de recomendación con IA: Consolida pruebas + perfil + presupuesto y recomienda programas SOLO del portafolio Grasshopper
✓ Integración O*NET Online: Muestra profesiones afines y salarios esperados por área
✓ Panel B2B para Colegios (CRÍTICO): Portal del Consejero donde el psicólogo ve todos sus estudiantes, qué pruebas han hecho, qué les dice la herramienta, qué decisión están tomando
✓ Integración con Bitrix CRM: Los usuarios registrados en Hopper que cumplan ciertos criterios se envían automáticamente como leads a Bitrix CRM con sincronización de información clave (nombre, email, perfil, intereses, programas recomendados)
✓ Base de datos de programas: Sistema para cargar y actualizar el portafolio de instituciones Grasshopper
✓ Documentación técnica, capacitación (2 horas) y 30 días de soporte post-entrega

❌ Lo que NO incluye:

✗ Chatbot perfilador en la web pública (6 preguntas)
✗ Dashboard de métricas del chatbot
✗ Panel de configuración del chatbot
✗ Bot de búsqueda interna para asesores
02

Hopper + Chatbot Perfilador + Bitrix

El embudo completo · Mejor ROI

$22.134.000
Fee de Implementación: $18.600.000 COP + IVA (19%)
Forma de pago: 50% anticipo + 50% contra entrega
Timeline: 12 semanas
Fee Mensual de Operación
$2.618.000 COP/mes
($2.200.000 + IVA · Sin permanencia mínima)

✅ TODO lo del Escenario 1 + lo siguiente:

✓ Chatbot Perfilador Completo (6 preguntas): Embebido en www.grasshopperstudy.com con IA conversacional
✓ Motor de análisis con IA: Procesa las 6 respuestas, genera perfil básico personalizado y califica al usuario (Premium, Potencial, Exploratorio, No Fit)
✓ CTA dinámico según perfil: Mensajes personalizados que invitan a registrarse en Hopper según el score del usuario
✓ Panel de configuración del chatbot: Modificar preguntas sin tocar código, ajustar respuestas y mensajes, cambiar lógica de calificación
✓ Dashboard de Métricas del Chatbot: Visualización en tiempo real de conversaciones iniciadas vs completadas, distribución por categoría, tasa de conversión Chatbot → Hopper, tiempo promedio de interacción, identificación de preguntas con más abandono, exportación a Excel
✓ Integración Chatbot → Hopper: Redirección automática, pre-carga de respuestas en perfil de Hopper, tracking de conversión, leads Premium marcados como prioritarios
✓ Integración avanzada con Bitrix CRM: Tanto los leads del chatbot como los usuarios registrados en Hopper se sincronizan con Bitrix automáticamente
🎯 Por qué es el escenario recomendado:
  • Inversión de $18.6M COP (dentro del presupuesto razonable)
  • Incluye el embudo completo: Chatbot gratuito → Hopper premium → Bitrix CRM
  • El chatbot es crítico para generar tráfico hacia Hopper (sin él, dependen 100% de marketing)
  • Dashboard de métricas permite medir conversión y optimizar el funnel en tiempo real
  • Panel B2B para colegios incluido (crítico para ventas B2B)
  • Integración con Bitrix CRM permite gestión centralizada de todos los leads
  • Timeline razonable: 3 meses para tener todo funcionando
