## Aprendizajes para el pipeline

Este caso revela algo sobre nuestro ICP:

- **Agencias en fase de arranque pueden no tener budget para AI Ops $7M**

- Para este perfil, **AI Edu Layer** puede ser el entry point mejor
- Considerar en futuras prospecciones: **filtrar mejor por revenue/tamaño** antes de presentar la oferta Ops

Si otras agencias emergentes se caen por la misma razón, validaremos que el filtro de ICP (empresas $250k–$3M USD revenue, 10–60 empleados) está bien calibrado — Renata probablemente estaba por debajo de ese umbral.
