Setup — Desktop App
PAG 08

Instalar Claude Desktop App

Esta es la opcion recomendada para la mayoria de usuarios. Se instala como cualquier otra aplicacion en tu computador. No necesitas saber nada tecnico.

Antes de empezar, necesitas

Cuenta en claude.aiCon plan Pro ($20 USD/mes) o superior. Creala en claude.ai si no la tienes.
ComputadorMac (macOS 13 o superior) o Windows (10 o superior)

Paso a paso

1
Descarga la app Claude

Abre tu navegador (Chrome, Safari, etc.) y ve a claude.ai. Ahi veras la opcion para descargar la app de escritorio. Haz clic en el boton de descarga para tu sistema (Mac o Windows).

En Mac el archivo descargado termina en .dmg — haz doble clic y arrastra Claude a tu carpeta de Aplicaciones. En Windows termina en .exe — haz doble clic y sigue el instalador.
2
Abre Claude e inicia sesion

Busca "Claude" en tus aplicaciones y abrela. La app te va a pedir que inicies sesion con tu correo y contrasena de claude.ai. Usa las mismas credenciales que usas en la pagina web.

3
Haz clic en la pestana "Code"

Arriba de la app vas a ver tres pestanas. Haz clic en la que dice "Code". Esa es la que activa los skills.

Chat Cowork Code ← esta!
Si te pide actualizar tu plan o pagar, significa que necesitas activar tu suscripcion Pro en claude.ai primero.

Instalar Claude Code en Terminal

Esta seccion es para perfiles tecnicos. Si ya instalaste la Desktop App en la pagina anterior, puedes saltarte esta pagina completamente. Esta opcion es identica en funcionalidad, solo cambia la interfaz (texto en vez de visual).

Instalacion rapida

1
Ejecuta el instalador

Abre tu terminal y copia este comando:

# Mac, Linux o WSL curl -fsSL https://claude.ai/install.sh | bash # Windows (PowerShell) irm https://claude.ai/install.ps1 | iex
En Windows se requiere Git for Windows. Instalalo desde git-scm.com si no lo tienes.
2
Navega a tu proyecto e inicia
cd /ruta/a/tu/carpeta-de-proyecto claude

La primera vez se abre tu navegador para autenticarte con tu cuenta de claude.ai. Despues de eso, ya puedes usar todos los skills con /nombre-del-skill.

3
Verifica que funciona

Escribe /que-sigue en la sesion de Claude Code. Si ves una respuesta de Claude, todo esta listo. Claude Code se actualiza automaticamente.

Ejemplo real: generar una cotizacion

1
Abre Claude y asegurate de estar en la pestana Code con tu carpeta de proyecto seleccionada.

Revisa el resultado en el chat. Si algo no te gusta, simplemente dilo: "cambia el precio a dolares" o "agrega 10% de descuento". Claude ajusta al instante.

Tips para sacarle provecho

Arrastra archivos al chatPuedes arrastrar PDFs, notas, transcripts, o cualquier archivo para darle contexto a Claude. Esto es muy util antes de ejecutar un skill.
Usa /que-sigue cuando estes perdidoEste es tu GPS. Te dice exactamente donde estas en el proceso y que skill ejecutar ahora.
Habla normalNo necesitas usar lenguaje especial. Di "hazlo mas corto", "cambia el tono", "ponlo en ingles" — Claude entiende perfectamente.
Usa @archivo para dar contextoEscribe @nombre-archivo en el chat para que Claude lea un archivo especifico sin tener que arrastrarlo.
Los archivos se organizan solosCada skill guarda sus resultados en la carpeta correcta del cliente automaticamente. No tienes que mover nada.
Interrumpe si va por mal caminoSi Claude no va bien, escribe tu correccion y presiona Enter. No necesitas esperar a que termine.

Preguntas Frecuentes
PAG 17

FAQ

"Escribi un comando con / pero no me reconoce el skill"
Verifica que estas en la carpeta correcta del proyecto. Los skills solo funcionan cuando Claude tiene acceso a la carpeta que te entrego Irrelevant. En la Desktop App, revisa que seleccionaste la carpeta correcta en "Select folder".
"Claude me dice que no tiene acceso o me pide autorizacion"
Claude te pide permiso antes de hacer cambios en tus archivos. Esto es una medida de seguridad. Revisa lo que propone hacer y dale permiso si estas de acuerdo. Si dice que tu plan no tiene acceso, verifica que tu suscripcion de Claude este activa.
"El resultado no es lo que esperaba"
Simplemente pidele cambios en lenguaje natural. Claude entiende instrucciones como "hazlo mas corto", "cambia el tono a mas formal", "agrega una seccion sobre X", "usa dolares en vez de pesos". No necesitas empezar de cero.
"No se cual skill usar para lo que necesito"
Escribe /que-sigue y Claude analiza en que punto estas del proceso y te recomienda el skill adecuado. Tambien puedes consultar la tarjeta de referencia rapida en la pagina 15 de esta guia.
"Quiero hacer algo para lo que no hay un skill"
Claude puede ayudarte con muchas tareas aunque no haya un skill especifico. Escribele lo que necesitas en lenguaje natural (sin usar /) y Claude hara su mejor esfuerzo. Si es algo que necesitas frecuentemente, contacta a Irrelevant para crear un skill personalizado.
"Cuanto cuesta la suscripcion de Claude?"
Los planes se pagan directamente a Anthropic (la empresa que hace Claude). Pro: $20 USD/mes — suficiente para la mayoria de usuarios. Max: $100-200 USD/mes — para uso intensivo. La suscripcion es independiente del servicio de Irrelevant.
"Se me cerro la sesion o la app"
Vuelve a abrir la app, inicia sesion, selecciona la pestana Code y tu carpeta de proyecto. Todos tus archivos y skills siguen intactos — nada se pierde.
"Puedo usar esto desde el celular?"
Actualmente Claude Code (los skills) solo funciona en computador (Mac o Windows). Puedes usar claude.ai desde el celular para chat normal, pero los skills requieren la Desktop App o el CLI en un computador.
