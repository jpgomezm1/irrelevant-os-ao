## PARÁMETROS GENERALES

### Unidades de Medida Válidas

- **Metro cuadrado (m²):** Superficies (estructuras, tanques)
- **Metro lineal (m):** Longitudes (soldadura, perfiles)
- **Kilogramo (Kg):** Pesos (metal, producto)
- **Unidad:** Producto completo (tanque, puerta, marco)
- **Hora:** Servicios por tiempo
- **Día:** Servicios por jornada (8 horas)
- **Servicio:** Trabajo genérico

### IVA por Grupo

Todos los productos/servicios usan **IVA 19%** (estándar para metalmecánica).

**Excepciones (si aplican):**
- Servicios de capacitación: 5% (reducida)
- Ciertas consultoría: 19%

Para este catálogo, asumir **19% como default**.

## CÓMO USAR ESTE CATÁLOGO

### En Remisión

Cuando Carolina reciba remisión de Jenguar:

**Paso 1:** Lee descripción del producto en remisión
**Paso 2:** Busca en este catálogo (por palabra clave)
**Paso 3:** Asigna código y cuenta automáticamente

**Ejemplo:**

```
Remisión dice: "Soldadura 20m lineales acero"
Búsqueda: "SOL-EST-001" ← Soldadura SMAW
Clasificación automática: 413545 (Soldadura), IVA 19%
```

### Si No Encuentra Exacto

Si descripción no encaja exactamente:

1. Busca el **grupo más cercano** (ej: "soldadura" → Grupo 413545)
2. Pregunta a Carolina: *"¿Es SOL-EST-001 o SOL-EST-002?"*
3. Registra nuevo código si es producto nuevo

## VALIDACIÓN

Antes de asignar cuenta, verificar:

- ¿La descripción encaja con algún producto del catálogo?
- ¿La unidad de medida es válida?
- ¿El precio es razonable?
- ¿La cuenta PUC existe en el PUC de Galco?

Si falla → marcar para revisión de Carolina.

## NOTAS IMPORTANTES

- **Precios estándar son de referencia 2026.** Pueden variar según cliente/proyecto.
- **Nuevos productos:** Si aparece algo no catalogado, agregarlo aquí y a la base maestra.
- **Descuentos:** Se aplican post-factura, no en cálculo de ingreso.
- **IVA:** Siempre 19% en factura. Cliente regimen no afecta tarifa, sólo retención.
