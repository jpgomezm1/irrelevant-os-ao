## CÓMO USAR ESTE DIRECTORIO

### En Remisión

Cuando recibas remisión con cliente:

**Paso 1:** Lee nombre del cliente
**Paso 2:** Busca en esta tabla
**Paso 3:** Si encuentras → auto-completa NIT, régimen, dirección
**Paso 4:** Si NO encuentras → preguntar NIT a Carolina

**Ejemplo:**

```
Remisión dice: "Constructora El Poblado"
Búsqueda: CLIENTE 1
Resultado: NIT 900567890, Responsable IVA
→ Auto-completa en factura
```

## VALIDACIÓN

Antes de usar cliente de esta lista:

- ¿El nombre es exacto?
- ¿El NIT es válido? (9 dígitos + DV)
- ¿El régimen es vigente?

Si hay discrepancia → **preguntar a Carolina.**
