# Seguimiento — IndemnEasy

**Cadencia:** Semanal.

## Convención de archivos

Cada reporte semanal se guarda como:
```
reporte-semanal-YYYY-MM-DD.md   (o .html)
```

La fecha corresponde al **cierre de la semana del reporte** (viernes, típicamente).

## Qué contiene cada reporte
- Avance de los 3 productos (Agente de Ventas, Agente Jurídico, CRM).
- Sprint activo y entregables completados.
- Bloqueos y riesgos.
- Decisiones pendientes del cliente.
- Próximos pasos.

## Herramienta
`/weekly-update indemneasy` genera el reporte usando el contexto del cliente.
